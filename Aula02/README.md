## Atividade - Aula 03 - API
Configurar uma API de Filmes e rodar o projeto em um dispositivo móvel.  <br>

# Desafio 1 
A primeira coisa para conectarmos a API dentro do nosso sistema foi entrar no site "TMDB" e gerar uma chave API. <br> 
Depois de gerada coloquei essa chave dentro do sistema no arquivo: .env  <br>
Com isso nosso APP já reconheceu a API trazendo filmes e informações <br>

# Desafio 2 
O desafio 2 foi criar um emulador no Android Studio para simularmos o projeto em um dispositivo Mobile. <br>
Outra parte dessa desafio era rodar o APP no celular, para isso ativei o modo de desenvolvedor e permite depuração USB e assim que conectei no projeto o APP já estava rodando perfeitamente no meu celular.
