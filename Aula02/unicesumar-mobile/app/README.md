# app

Projeto Flutter principal da disciplina.

## Objetivo

Concentrar os exemplos de construção de interface e organização de aplicações trabalhados em aula.

## Executar o Projeto

```bash
flutter pub get
flutter run
```

## Executar Testes

```bash
flutter test
```

## Organizacao

- `lib/`: código-fonte da aplicação.
- `test/`: testes de widget/unidade.
- `docs/`: roteiros e materiais de apoio da aula.
- `aula_widget/`: projeto complementar para foco em widgets/layouts.

## Recomendações para Aula

- Execute `flutter doctor` antes de iniciar.
- Utilize commits pequenos por tópico/aula.
- Evite versionar arquivos gerados automaticamente (build/cache).
